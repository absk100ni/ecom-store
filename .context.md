## BACKEND API (Running at http://localhost:8080)

### Public APIs
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/products?page=1&limit=12&category=&search=` | Products with pagination |
| GET | `/api/v1/products/:id` | Product detail (also works with slug) |
| GET | `/api/v1/categories` | List categories |
| POST | `/api/v1/auth/send-otp` | `{phone}` → sends OTP (dev: always 123456) |
| POST | `/api/v1/auth/verify-otp` | `{phone, code}` → returns JWT + user |

### Protected APIs (need `Authorization: Bearer <token>`)
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/cart` | Get cart `{cart, total, item_count}` |
| POST | `/api/v1/cart/add` | `{product_id, quantity}` |
| PUT | `/api/v1/cart/item/:productId` | `{quantity}` |
| DELETE | `/api/v1/cart/item/:productId` | Remove item |
| DELETE | `/api/v1/cart` | Clear cart |
| POST | `/api/v1/orders` | `{shipping_address, coupon_code?}` (creates from cart) |
| GET | `/api/v1/orders` | List user orders |
| GET | `/api/v1/orders/:id` | Order detail |
| POST | `/api/v1/payment/create` | `{order_id}` → Razorpay order |
| POST | `/api/v1/payment/verify` | Verify payment |
| GET | `/api/v1/shipping/track/:orderId` | Track shipment |
| GET | `/api/v1/user` | User profile |
